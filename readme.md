### Installation

```sh
git clone https://github.com/pqml/bretagne
cd bretagne
npm install
```

<br>
<h1></h1>
<br>

### Usage
<br>

#### `npm run start`
Run the development server.

#### `npm run build`
Compile a production-ready build into the dist folder.

#### `npm run build:serve`
Serve and test the current compiled website before deployment.
